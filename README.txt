INSTRUCCIONES PARA USAR EL CHATBOT GASTRONÓMICO

1. Instala Flask si no lo tienes:
   pip install flask

2. Coloca los archivos en la misma carpeta.

3. Ejecuta el servidor:
   python bot_backend.py

4. Abre el navegador en:
   http://127.0.0.1:5000

5. Escribe una región como "Costa", "Sierra" o "Selva" para ver los platos.

Nota: Asegúrate de tener MySQL corriendo y la base de datos 'gastronomia_peru' con las tablas y datos creados.
